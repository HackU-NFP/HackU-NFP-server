# すまーとみんと Server

## 起動方法
```bash
docker-compose up -d
```
Open localhost:8080


## 
webhook
```
/linebot/callback
```
nft一覧の取得
```
api/nfts
```
nftの詳細取得
```
api/nft
```
ユーザーのnft取得
```
/api/users/:userId/nfts
```
